Octo-Web is a prototype for an interface for Octosuite developed for Bellingcat's April 2023 hackathon.

Octosuite is a framework for gathering open-source intelligence on Github users,repositories and organisations.

The original tool was developed by Richard Mwewa. Octo-Web is based on his sterling work.

If you want to read further you should visit the Github repository here.

Special thanks also goes to Kate Morley whose content I relied on to remain faithful to Octosuite's tree rendering.
